# Tasks Client

Tasks API client.
* [Tasks API Repository](https://github.com/AlexRamos93/tasks-api).

## Getting Started

Clone or download the repository.

### Prerequisites

Your need the Tasks API running.

### Installing
```
$ npm install -g
```

### Running
```
$ npm start
```

## Built With

* [Javascript](https://www.javascript.com/) - The language used.
* [Node.JS](https://nodejs.org/)
